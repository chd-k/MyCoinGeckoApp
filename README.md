# MyCoinGeckoApp

### Description

There is an app that show crypto data provided by [CoinGecko API](https://www.coingecko.com/)

### Technology stack

- Retrofit
- OkHttp
- Coil
- Coroutines
- Flow
- ViewModel
- View binding
- MVVM
- Material Design

### Process decomposition

1. Setting up Gradle
2. Setting up layers

4. Setting up responses, API and repositories
5. Setting up ViewModels
6. Setting up RecyclerView
7. Refactoring
8. Testing
